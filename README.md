# 🛒 E-Commerce — Java Spring Boot & ReactJS

A full-stack E-Commerce web application built with **Java Spring Boot** (Backend) and **ReactJS** (Frontend).

---

## 📂 Repository Structure

This repository uses **separate branches** for backend and frontend:

| Branch | Tech Stack | Description |
|--------|-----------|-------------|
| [`main`](https://github.com/Tuanhung0912/E-Commerce-Java-Spring-Boot-And-ReactJS/tree/main) | Java, Spring Boot, Maven | REST API, Authentication, Database |
| [`frontend`](https://github.com/Tuanhung0912/E-Commerce-Java-Spring-Boot-And-ReactJS/tree/frontend) | React, Vite, TailwindCSS | User Interface, SPA |

### 🚀 Getting Started

#### Backend (branch `main`)
```bash
git clone -b main https://github.com/Tuanhung0912/E-Commerce-Java-Spring-Boot-And-ReactJS.git
cd E-Commerce-Java-Spring-Boot-And-ReactJS
./mvnw spring-boot:run
